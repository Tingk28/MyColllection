使用說明：

本系統已經排除使用者輸錯大小寫導致程式錯誤的問題，但仍然麻煩確保輸入指令格式正確，如add應該以半形空格隔開，以免程式要求您重新輸入。

進入本系統後，您會先看到本餐廳的菜單。
如需要指令集可以直接輸入”help”查看，（如下所示）
=================================================
1.add 品名 大小 甜度 冰塊 份數
新增咖啡至餐點明細中。以下為新增2杯大杯卡布奇諾半糖少冰至明細中。
add Cappuccino large half less 2

2.check
顯示目前所有的餐點明細，包含 編號、飲品、大小、甜度、 冰塊、份數、價格以及熱量。

3.edit 編號 項目 內容
可編輯某一則餐點明細中的 大小(size)、甜度(sugar)、冰 塊(ice)、份數(quantity)。以下為將第一則餐點中的大小改為medium。
edit 1 size medium

4.menu
顯示可選購的飲品，包含價 格及熱量。

5.help
列出所有指令集，並說明各 個指令的功能。

6.delete
刪除該次點餐紀錄。例如刪除第三筆
delete 3

7.bill
進行結帳，列出所有餐點明細和總價

===============================================
1.	依據您需要的餐點一一輸入及添加指令。如需更改請使用edit功能，如要移除則使用delete功能。

2.	點餐完成後輸入bill進行結帳
